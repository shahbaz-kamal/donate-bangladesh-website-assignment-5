Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. The DOM represents each elements / tag / node as documents or objects. DOM gives many meaningful ways to interact a webpage, manipulating the webpage by making the webpage dynamic.

How do you select an element from the DOM?

There are many ways of selecting one or elements. Some of them are given below:
1.document.getElementById(): Inside the first bracket the id of an element should be given . For example this <button> id="history-btn"</button> this tag should be targeted by document.getElementById(“history-btn”). This results in giving one element as ids are unique .
2.document.getElementsByClassName():This gives all the elements that have the class mentioned. For Example <p>class=”section-description”> </p>, this will give all the elements with class name section-description.
3.document.querySelector(): This targets the first element that matches the CSS selector.
4.document.querySelectorAll(): This targets the all element that matches the CSS selector.
5.document.getElementsByTagName(): This target the elements by tag name. Gives all elements having specific tag name.
There are many other ways of selecting one or elements.

What is event delegation in the context of the DOM, and why is it useful?

To handle Events at a higher level in DOM than the elements where the event originated a technique called event delegation is used. This technique uses event propagation & event bubbling. In simple way, in this technique we can manage events for a child element by simply adding an event listener to its parent element.

How do you manipulate an element's attributes and styles using the DOM?

1. Manupulating an Element's attributes: First We need to target the element by some ways like document.getElementById() and than adding .classList.add/remove("name-of class") . For Example , document.getElementById("history-tab").classList.add("hidden"),hide the section having id "history-tab". There are more ways of manupulating attributes.
2. Manupulating styles: Basic format is element.style.propertyName = 'value';.For Example, document.getElementById("history-tab").style.background='green', will make a green background for the section having ID "history-tab"
